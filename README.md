🛒 Blinkit Sales Dashboard (Power BI) 📊

📌 Project Overview

This project presents an interactive Blinkit Sales Dashboard built using Microsoft Power BI to analyze quick-commerce retail performance and business trends.

The dashboard provides insights into sales, product categories, outlet performance, and customer preferences, helping businesses make data-driven decisions.
